## Overview

PowerShell is a legitimate Windows command-line shell and scripting language that is frequently abused by attackers to execute commands, download payloads, perform reconnaissance, and automate malicious activities. Because PowerShell is widely used by both administrators and attackers, monitoring its execution is an important component of endpoint detection.

This detection demonstrates how PowerShell execution can be identified using Sysmon process creation telemetry and validated within Splunk Enterprise after simulating attacker behaviour with Atomic Red Team.

---

## MITRE ATT&CK

| Field | Value |
|--------|-------|
| Technique ID | T1059.001 |
| Technique | PowerShell |
| Tactic | Execution |

---

## Detection Objective

The objective of this detection is to identify PowerShell execution on a Windows endpoint using Sysmon Event ID 1 (Process Creation). By monitoring PowerShell process creation, defenders can detect suspicious script execution and other attacker activity that relies on PowerShell.

---

## Attack Simulation

The Atomic Red Team framework was used to simulate PowerShell execution by running the following Atomic Test:

```powershell
Invoke-AtomicTest T1059.001 -TestNumbers 17
```

**Expected Result**

- PowerShell process is executed.
- Sysmon generates Process Creation (Event ID 1).
- The event is forwarded to Splunk Enterprise.
- The detection query identifies the activity.

---

## Telemetry Generated

| Source | Event |
|---------|-------|
| Sysmon | Event ID 1 – Process Creation |

The primary telemetry used for this detection was Sysmon Process Creation events, which capture the executed process, command line, parent process, user, and execution time.

---

## SPL Detection Query

```spl
index=* "powershell.exe"
| rex field=_raw "<Data Name='Image'>(?<Image>[^<]+)</Data>"
| rex field=_raw "<Data Name='CommandLine'>(?<CommandLine>[^<]+)</Data>"
| rex field=_raw "<Data Name='ParentImage'>(?<ParentImage>[^<]+)</Data>"
| rex field=_raw "<Data Name='ParentCommandLine'>(?<ParentCommandLine>[^<]+)</Data>"
| rex field=_raw "<Data Name='User'>(?<User>[^<]+)</Data>"
| table _time User Image CommandLine ParentImage ParentCommandLine
| sort -_time
```

---

## Detection Validation

The detection successfully identified PowerShell execution generated during the Atomic Red Team simulation.

Validation confirmed:

- PowerShell process execution
- Command line arguments
- Parent process relationship
- Executing user
- Event timestamp
